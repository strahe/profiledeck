# ProfileDeck

Safe profile switching for AI agents and coding tools.

ProfileDeck saves local AI coding tool logins and settings as Profiles, then lets you review and apply a switch when you need a different setup.

## Get started

- **Desktop Alpha:** [Download for macOS 14+ on Apple silicon](https://github.com/strahe/profiledeck/releases)
- **CLI:** [Build from source](docs/guide/getting-started.md#build-and-use-the-cli)

## Supported tools

| Tool | What you can switch |
| --- | --- |
| Codex | Login and settings |
| Claude Code | Subscription login |
| Antigravity | agy v2 login |

You can also view local Codex usage and check usage limits. See [supported tools and limitations](docs/index.md#supported-tools).
